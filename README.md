# SGCarsTrends API

[![Quality Gate Status](https://sonarcloud.io/api/project_badges/measure?project=sgcarstrends_api&metric=alert_status)](https://sonarcloud.io/summary/new_code?id=sgcarstrends_api)

## Overview

SGCarsTrends API provides data and insights about car trends in Singapore.

## Getting Started

### Prerequisites

- Node.js
- pnpm

### Installation

1. Clone the repository
2. Install dependencies
   ```bash
   pnpm install
   ```
3. Run the application
   ```bash
   pnpm dev
   ```

## Development

- Run tests
  ```bash
   pnpm test
   ```

- Lint code
  ```bash
   pnpm biome lint
   ```

- Typecheck
  ```bash
   pnpm typecheck
   ```


- Deploy
  ```bash
   pnpm deploy
   ```

- Remove deployment
  ```bash
   pnpm remove
   ```

- Open SST Console
  ```bash
   pnpm console
   ```
